Top 20 des Fonctions Excel (0/20)

1. SOMME (SUM)

Additionne tous les nombres dans une plage de cellules.

Syntaxe :

=SOMME(nombre1; [nombre2]; ...)

Exemple :

Pour additionner les valeurs des cellules A1 à A10 :

=SOMME(A1:A10)

2. MOYENNE (AVERAGE)

Calcule la moyenne arithmétique d'une plage de cellules.

Syntaxe :

=MOYENNE(nombre1; [nombre2]; ...)

Exemple :

Pour la moyenne des notes en A1:A10 :

=MOYENNE(A1:A10)

3. SI (IF)

Effectue un test logique et renvoie une valeur si VRAI, et une autre si FAUX.

Syntaxe :

=SI(test_logique; valeur_si_vrai; valeur_si_faux)

Exemple :

Si la note en A1 est > 10, affiche "Réussi", sinon "Échec" :

=SI(A1>10; "Réussi"; "Échec")

4. RECHERCHEV (VLOOKUP)

Recherche une valeur dans la première colonne d'un tableau et renvoie une valeur dans la même ligne d'une colonne spécifiée.

Syntaxe :

=RECHERCHEV(valeur_cherchée; table_matrice; no_index_col; [valeur_proche])

Exemple :

Cherche A1 dans la plage C1:D10, et renvoie la 2ème colonne (correspondance exacte) :

=RECHERCHEV(A1; C1:D10; 2; FAUX)

5. NB.SI (COUNTIF)

Compte le nombre de cellules dans une plage qui répondent à un critère.

Syntaxe :

=NB.SI(plage; critère)

Exemple :

Compte le nombre de fois que "Oui" apparaît dans la plage A1:A10 :

=NB.SI(A1:A10; "Oui")

6. SOMME.SI (SUMIF)

Additionne les cellules d'une plage qui répondent à un critère.

Syntaxe :

=SOMME.SI(plage; critère; [somme_plage])

Exemple :

Additionne les montants en B1:B10 si la catégorie en A1:A10 est "Fruits" :

=SOMME.SI(A1:A10; "Fruits"; B1:B10)

7. RECHERCHEX (XLOOKUP)

Moderne et remplace RECHERCHEV. Recherche une valeur et renvoie un résultat de n'importe quelle colonne.

Syntaxe :

=RECHERCHEX(valeur_cherchée; tableau_recherche; tableau_renvoyé; ...)

Exemple :

Cherche A1 dans la colonne C et renvoie la valeur de la colonne D :

=RECHERCHEX(A1; C:C; D:D)

8. MAX

Renvoie la plus grande valeur d'une plage de cellules.

Syntaxe :

=MAX(nombre1; [nombre2]; ...)

Exemple :

Trouve le montant le plus élevé en A1:A10 :

=MAX(A1:A10)

9. MIN

Renvoie la plus petite valeur d'une plage de cellules.

Syntaxe :

=MIN(nombre1; [nombre2]; ...)

Exemple :

Trouve le montant le plus bas en A1:A10 :

=MIN(A1:A10)

10. NB (COUNT)

Compte le nombre de cellules qui contiennent des nombres.

Syntaxe :

=NB(valeur1; [valeur2]; ...)

Exemple :

Compte combien de cellules en A1:A10 contiennent des chiffres :

=NB(A1:A10)

11. NBVAL (COUNTA)

Compte le nombre de cellules qui ne sont pas vides (contiennent du texte, des nombres, etc.).

Syntaxe :

=NBVAL(valeur1; [valeur2]; ...)

Exemple :

Compte le nombre de cellules non vides en A1:A10 :

=NBVAL(A1:A10)

12. ET (AND)

Vérifie si TOUTES les conditions sont VRAIES. Renvoie VRAI ou FAUX.

Syntaxe :

=ET(logique1; [logique2]; ...)

Exemple :

Souvent utilisé avec SI. Vérifie si A1 > 10 ET B1 = "Oui" :

=SI(ET(A1>10; B1="Oui"); "VRAI"; "FAUX")

13. OU (OR)

Vérifie si AU MOINS UNE condition est VRAIE. Renvoie VRAI ou FAUX.

Syntaxe :

=OU(logique1; [logique2]; ...)

Exemple :

Vérifie si A1 > 10 OU B1 = "Oui" :

=SI(OU(A1>10; B1="Oui"); "VRAI"; "FAUX")

14. CONCAT

Combine le texte de plusieurs cellules en un seul. (Remplace CONCATENER).

Syntaxe :

=CONCAT(texte1; [texte2]; ...)

Exemple :

Combine le prénom en A1 et le nom en B1, séparés par un espace :

=CONCAT(A1; " "; B1)

15. GAUCHE (LEFT)

Extrait un nombre de caractères en partant de la gauche d'un texte.

Syntaxe :

=GAUCHE(texte; [no_car])

Exemple :

Extrait les 3 premiers caractères du texte en A1 :

=GAUCHE(A1; 3)

16. DROITE (RIGHT)

Extrait un nombre de caractères en partant de la droite d'un texte.

Syntaxe :

=DROITE(texte; [no_car])

Exemple :

Extrait les 2 derniers caractères du texte en A1 :

=DROITE(A1; 2)

17. STXT (MID)

Extrait un nombre de caractères d'un texte à partir d'une position spécifiée.

Syntaxe :

=STXT(texte; no_départ; no_car)

Exemple :

Extrait 4 caractères du texte en A1, en commençant au 3ème caractère :

=STXT(A1; 3; 4)

18. AUJOURDHUI (TODAY)

Renvoie la date du jour actuelle. Se met à jour à l'ouverture du fichier.

Syntaxe :

=AUJOURDHUI()

Exemple :

Affiche la date actuelle dans la cellule :

=AUJOURDHUI()

19. SIERREUR (IFERROR)

Renvoie une valeur spécifiée si une formule génère une erreur (ex: #DIV/0!).

Syntaxe :

=SIERREUR(valeur; valeur_si_erreur)

Exemple :

Si A1/B1 est une erreur, affiche "0", sinon affiche le résultat :

=SIERREUR(A1/B1; 0)

20. INDEX + EQUIV

Une combinaison puissante. EQUIV trouve la position d'une valeur, et INDEX la récupère dans une autre colonne.

Syntaxe :

=INDEX(plage_résultat; EQUIV(valeur_cherchée; plage_recherche; 0))

Exemple :

Cherche A1 dans la colonne C et renvoie la valeur de la colonne D (comme RECHERCHEX) :

=INDEX(D:D; EQUIV(A1; C:C; 0))
