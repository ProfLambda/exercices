Guide Pratique : Votre "Dictionnaire Métier"

spellcheck Auto-Correction

Automatisez les termes complexes (ex: "brady" -> "bradycardie").

  1. 1. Fichier > Options > Vérification
  2. 2. Options de correction automatique
  3. 3. Tapez le code dans "Remplacer"
  4. 4. Tapez le mot complet dans "Par"

text_snippet QuickParts

Insérez des paragraphes entiers ou des conclusions types.

  1. 1. Sélectionnez le texte
  2. 2. Insertion > Explorer QuickPart
  3. 3. Enregistrer la sélection
  4. 4. Appelez via le nom + Touche F3

database Dictionnaire .dic

Évitez le soulignement rouge des termes médicaux valides.

  1. 1. Clic droit sur un mot souligné
  2. 2. Ajouter au dictionnaire
  3. 3. Exportez le fichier .dic pour vos autres postes de travail
