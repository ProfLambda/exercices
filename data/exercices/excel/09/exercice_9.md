# Brief de l’exercice – Suivi de l’ancienneté du personnel dans un secrétariat médical

## Contexte professionnel

Vous travaillez comme **secrétaire médical(e)** dans un centre de santé regroupant plusieurs professionnels.  
Le responsable administratif vous confie un tableau contenant les informations de plusieurs membres du personnel arrivés à différentes dates dans l’établissement.

Votre mission est de **compléter et exploiter ce tableau** afin de faciliter le suivi administratif des agents.

À partir des colonnes déjà renseignées :

- **Prénom**
- **Nom**
- **début**
- **fin**

vous devez calculer :

- **l’ancienneté** de chaque agent sous une forme lisible : `X an(s) et Y mois`
- **l’ancienneté en mois**
- **l’ancienneté en année**

## Utilité dans le métier

Dans un secrétariat médical, ce type de tableau peut être utilisé pour :

- suivre l’ancienneté du personnel administratif ou soignant ;
- préparer certains documents RH ;
- vérifier des informations utiles pour les dossiers du personnel ;
- transmettre des données claires à la direction ou au service comptable.

Le résultat doit être **fiable, lisible et exploitable rapidement**, comme dans une situation professionnelle réelle.

## Objectifs de l’exercice

Cet exercice permet de travailler :

- la saisie et la lecture de données dans un tableau Excel ;
- l’utilisation d’une formule de calcul entre deux dates ;
- la présentation d’un résultat sous plusieurs formes ;
- l’automatisation d’un calcul sur l’ensemble d’un tableau.

## Travail demandé

À l’aide du tableau fourni, vous devez :

1. calculer l’**ancienneté rédigée** pour chaque personne ;
2. calculer l’**ancienneté totale en mois** ;
3. calculer l’**ancienneté totale en années** ;
4. recopier les formules sur toutes les lignes du tableau ;
5. vérifier que les résultats sont cohérents.

## Attendu professionnel

En tant que secrétaire médical(e), vous devez produire un tableau :

- propre ;
- exact ;
- facile à relire ;
- utilisable par un collègue ou un responsable sans avoir à refaire les calculs.

## Exemple attendu

Pour une personne ayant commencé le **27/10/2009** et terminé le **10/03/2026**, le tableau doit permettre d’obtenir :

- **16 an(s) et 4 mois**
- **196 mois**
- **16 années**
